# clima_app
Una aplicación móvil desarrollada en Flutter que permite consultar el clima actual de cualquier ciudad. Con una interfaz limpia y atractiva, muestra la temperatura, condiciones climáticas, y máximas y mínimas del día. La app cambia dinámicamente el fondo según el estado del clima, brindando una experiencia visual más realista.
